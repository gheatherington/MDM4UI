# Directory Overview

This directory serves as a repository for educational notes, specifically for a data management or statistics course. The content is stored in Markdown files and covers fundamental concepts in the field.

# Key Files

*   **`Unit 1.md`**: Contains notes on combinatorics, including permutations, combinations, factorials, and Pascal's Triangle. It provides formulas and examples for calculating arrangements and selections.
*   **`Unit 2.md`**: Focuses on probability theory. It covers theoretical vs. experimental probability, conditional probability, independent vs. dependent events, and the rule of sum. It also includes probability tree diagrams.
*   **`Unit 3.md`**: Details concepts related to data itself. This includes types of data (qualitative, quantitative), methods of data collection (surveys, experiments), survey question design, sampling techniques, and various forms of statistical bias.
*   **`Unit-3-Flashcards.md`**: A study aid generated from the `Unit 3.md` file. It contains flashcards in a format compatible with the Obsidian Spaced Repetition plugin, covering key terms, definitions, and examples from the unit.

# Usage

The primary purpose of this directory is for studying and knowledge management. The user, likely a student, uses these files to review course material.

Common tasks might include:
*   Asking for explanations of specific concepts found within the notes.
*   Requesting summaries of the content in one or more files.xx`
*   Generating new study aids, such as flashcards or quizzes, from the existing notes.
*   Answering questions based on the content of the markdown files.
